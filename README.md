# Modern Login Page

A sleek, modern login page with glassmorphism design, smooth animations, and responsive layout.

## Features

- **Glassmorphism Design**: Transparent login card with blur background effect
- **Dark Neon Theme**: Purple/blue gradient with animated background
- **Smooth Animations**: Hover effects, glowing inputs, and loading animations
- **Input Icons**: Username (👤) and password (🔒) icons
- **Password Toggle**: Show/hide password functionality
- **Loading Animation**: Spinner during login process
- **Error Handling**: Clear error messages for invalid credentials
- **Responsive Design**: Works on both desktop and mobile devices
- **Sound Effects**: Audio feedback on login attempts
- **Remember Me**: Checkbox for remembering login credentials

## Usage

1. Open `login.html` in a modern web browser
2. Use the default credentials:
   - Username: `admin`
   - Password: `password`
3. Click "Login" to authenticate
4. On successful login, you'll be redirected to Amazon.in

## Technologies Used

- HTML5
- CSS3 (with modern features like backdrop-filter)
- JavaScript (ES6+)

## Browser Support

Works best in modern browsers that support:
- CSS backdrop-filter
- CSS animations
- Web Audio API

## Customization

You can easily customize:
- Colors and gradients in the CSS
- Login credentials in the JavaScript
- Redirect URL after successful login
- Animation timings and effects
